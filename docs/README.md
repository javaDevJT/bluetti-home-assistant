# Bluetti Enhance Documentation

This directory stores project knowledge gathered while enhancing the official
BLUETTI Home Assistant custom integration. Keep durable findings here instead of
leaving them only in chat or tool output.

## Structure

- `release/` - release and distribution notes such as HACS packaging
  requirements and publication checklists.

## Usage Notes

- Public release notes intentionally avoid private capture payloads, serial
  numbers, and account-specific diagnostics.
- Treat device support as data-driven unless proven otherwise.
